# cloud_music

```
cloud_music
├─ cloud_music_server_api // 后端 Api
├─ components // 组件
|     ├─NavHeader // 导航头部组件
├─ utils // 工具函数封装
|   ├─ config.js // 请求配置文件
|   ├─ request.js // 请求封装
├─ static // 静态资源
├─ songPackage // 歌曲详情页分包
|      ├─ pages
|      |   ├─ static // 详情页静态资源
|      |   ├─ songDetail
|      |   ├─ recommendSong
├─ pages // 主包
|   ├─ video // 视频页
|   ├─ search  // 搜索页
|   ├─ personal // 个人中心页
|   ├─ login // 登录页
|   ├─ index  // 首页
```

后端应用基于 [NeteaseCloudMusicApi](https://github.com/Binaryify/NeteaseCloudMusicApi) 搭建
